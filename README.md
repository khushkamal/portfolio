# 🚀 Personal Portfolio | Khushkamal Singh

A modern, high-performance personal portfolio built with React and Vite. This project features a clean dark-mode aesthetic, smooth animations, and a dedicated project detail routing system.

## ✨ Key Features

-   **Dynamic Project Routing**: Dedicated detail pages for each project using React Router v7.
-   **Smooth Animations**: Powered by Framer Motion for a premium look and feel.
-   **Modern Tech Stack**: React 19, Tailwind CSS 3.4, and Vite 7.
-   **Interactive UI**: Custom cursor tracking, particle backgrounds, and glassmorphism components.
-   **Clean Contact Flow**: Direct mail integration for seamless communication.

## 🛠️ Tech Stack

-   **Frontend**: React (v19)
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion, React Icons
-   **Routing**: React Router (v7)
-   **Build Tool**: Vite

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/khushkamal/portfolio.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```
